## 1. Growth Rates Benchmarks (SaaS B2C)

### O Que É "Bom" para Diferentes Estágios

| Estágio | MRR | Crescimento Mensal "Bom" | "Excelente" |
|---------|-----|--------------------------|-------------|
| Pre-PMF | <$10k | Qualquer crescimento com retenção | - |
| Early | $10k-100k | 15-20% MoM | 25%+ MoM |
| Growth | $100k-1M | 10-15% MoM | 20%+ MoM |
| Scale | $1M+ | 5-10% MoM | 15%+ MoM |

**Para CortAI agora (early stage):**
- Meta realista: 15-20% MoM de crescimento no número de pagantes
- Meta otimista: 25%+ MoM

### Regra do "T2D3" para SaaS B2B (adaptada para B2C)
Para B2C, o equivalente é dobrar a base de pagantes a cada 6-9 meses no early stage.

## 2. Retenção por Tipo de Produto

### Benchmarks de Retenção (D30 — usuários que voltaram)
| Tipo de Produto | Bom | Ótimo |
|----------------|-----|-------|
| Consumer app | 25% | 40%+ |
| SMB SaaS | 70% | 85%+ |
| B2C SaaS (como CortAI) | 30-40% | 50%+ |
| Productivity tools | 40% | 60%+ |

**Para CortAI:** Meta de 35%+ de retenção D30 (usuários que usaram o produto em pelo menos 2 semanas diferentes no mês)

### "The Smile Retention Curve"
```
100% |*
     |  *
 50% |     *  *  *  *  *  *  (curve flattens = PMF)
     |           *  *
  0% |________________________________
     D1  D7  D14  D30  D60  D90
```
Se a curva continua caindo até zero = sem PMF.
Se a curva achata = tem um core de usuários que ficam.

## 4. Activation Framework

### A Definição de Ativação de Lenny
"Ativação não é só o usuário se cadastrar. É o usuário chegar ao Aha Moment."

### Como Definir Seu Aha Moment
Analise: O que usuários retidos fazem nos primeiros 3 dias que usuários churned não fazem?

**Para CortAI — possíveis Aha Moments:**
- Submete primeiro vídeo + vê o primeiro clip gerado ✓
- Faz download do primeiro clip (ainda mais forte)
- Publica o primeiro clip nas redes (mais forte ainda)

### Improving Activation
1. **Medir o funil:** Cadastro → Vídeo submetido → Pipeline completo → Clip baixado
2. **Encontrar o dropout maior:** Onde mais usuários abandonam?
3. **Reduzir fricção no maior dropout**
4. **Testar:** Nova mensagem, novo tutorial, novo default

## 5. PLG (Product-Led Growth) para Ferramentas Criativas

### O Modelo de PLG
```
Usuário usa o produto → Produto entrega valor imediato →
Produto se vira viral ou de referência → Novos usuários chegam pelo produto
```

### PLG Tactics para CortAI
1. **Free tier generoso mas com limite claro** — Usuário usa, gosta, quer mais
2. **Frictionless onboarding** — Criar primeiro clip em <5 minutos
3. **Viral no produto** — Clips com marca d'água opcional
4. **Sharing nativo** — Botão "compartilhar clip" direto para TikTok/Instagram
5. **Invite mechanism** — "Indique 3 amigos, ganhe 1 mês grátis"

## 6. Pricing Best Practices (Lenny)

### O Erro #1 de Pricing
"A maioria das startups subestima quanto os usuários pagariam."

### Framework de Pricing
1. **Pesquise willingness to pay:** Pergunte diretamente a 20 usuários
2. **Anchore no valor entregue:** Quanto vale para o usuário criar 10 clips/mês?
3. **Teste diferentes preços:** A/B test de pricing page
4. **Aumente preços quando tem PMF:** Usuários com PMF pagam mais

### Para CortAI
- Um criador BR paga R$50/mês facilmente se o produto salva 10+ horas
- Um criador que ganha R$5k/mês com vídeos vê CortAI como investimento, não gasto
