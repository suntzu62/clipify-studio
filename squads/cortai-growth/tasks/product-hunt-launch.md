# Task: product-hunt-launch

Prepara e executa um lançamento no Product Hunt.

## Checklist de Preparação (4 semanas antes)

### Semana 4 — Fundação
- [ ] Criar conta no Product Hunt (se não tiver)
- [ ] Definir hunter (idealmente alguém com seguidores no PH)
- [ ] Escrever tagline (máx 60 chars): "Crie clips virais automaticamente com IA"
- [ ] Escrever descrição (máx 260 chars)
- [ ] Preparar screenshots (mínimo 3, ideal 5)
- [ ] Criar GIF demo ou vídeo curto (30-60 segundos)
- [ ] Definir oferta especial para usuários PH (desconto 30 dias?)

### Semana 3 — Conteúdo
- [ ] Escrever post "Maker Post" (história do produto, 200-400 palavras)
- [ ] Preparar resposta para perguntas frequentes esperadas
- [ ] Criar landing page especial para tráfego do PH
- [ ] Preparar email para lista existente de usuários/interessados

### Semana 2 — Comunidade
- [ ] Contatar early adopters para upvote no dia
- [ ] Engajar em comunidades BR (grupos de criadores) para divulgação
- [ ] Postar no LinkedIn/Twitter anunciando o lançamento iminente
- [ ] Pedir a 10 pessoas para testar e dar feedback antes do PH

### Semana 1 — Execução
- [ ] Agendar post para 00:01 PST (horário ideal)
- [ ] Preparar template de agradecimento para comentários
- [ ] Configurar monitoring do Sentry/PostHog para pico de tráfego
- [ ] Garantir que backend aguenta 10x tráfego normal
- [ ] Preparar respostas para primeiros 20 comentários prováveis

## Dia do Lançamento
- [ ] Postar às 00:01 PST
- [ ] Responder TODOS os comentários nas primeiras 4 horas
- [ ] Postar nas redes sociais (@content-creator)
- [ ] Enviar email para lista
- [ ] Monitorar upvotes e posição de hora em hora

## Métricas de Sucesso
- 200+ upvotes = resultado bom
- 500+ upvotes = resultado excelente
- 50+ sign-ups no dia = conversão boa
- Top 5 do dia = exposição máxima

## Copy Essencial
**Tagline (60 chars):** "Crie short clips virais de vídeos longos com IA"
**Headline da landing PH:** [escrever com @content-creator]
**Oferta especial:** [definir: 50% off primeiro mês, 3 meses grátis, etc.]
