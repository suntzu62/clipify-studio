# Task: squad-briefing — CEO Squad Alignment

## Objetivo
Preparar e entregar briefing estratégico do CEO para um ou todos os squads.

## Quando Usar
- Início de novo quarter com OKRs definidos
- Mudança de prioridade estratégica
- Após *pivot-or-persevere decision
- Após *wartime-mode activation

## Processo

### STEP 1 — Contexto do Briefing
Perguntar ao CEO (usuário):
1. O que mudou que motivou este briefing?
2. Qual é a mensagem central (em 1 frase)?
3. Qual squad(s) está recebendo o briefing?

### STEP 2 — Estrutura Universal do Briefing CEO

```
📢 BRIEFING CEO — [Data]
Para: [Squad / Todos os Squads]
De: Vega (CEO)

═══════════════════════════════════════
🌍 CONTEXTO ESTRATÉGICO (Por quê agora)
[Explicar o estado do negócio que motiva este briefing]
[Métricas relevantes que contextualizam]
[Ameaça ou oportunidade identificada]

🎯 PRIORIDADE #1 DESTE SQUAD ESTE SPRINT
[1 coisa específica. Apenas 1.]
[Por que esta é a prioridade? Qual KPI ela move?]

📊 MÉTRICAS QUE VOU MONITORAR
- [Métrica 1]: [valor atual] → [meta] até [data]
- [Métrica 2]: [valor atual] → [meta] até [data]

🚫 O QUE NÃO FAZER AGORA
[O que explicitamente pausar ou descartar para focar]

🔗 DEPENDÊNCIAS CROSS-SQUAD
- [O que este squad precisa de outro squad]
- [O que este squad entrega para outro squad]

📅 PRÓXIMA REVISÃO COM CEO: [data]
═══════════════════════════════════════
```

### STEP 3 — Briefings por Squad

**cortai-tech:**
- Contexto: impacto das decisões técnicas no produto e retenção
- Métricas CEO para tech: uptime, tempo de processamento, bugs críticos abertos, deploy frequency
- Pergunta-chave: "Qual é o maior bloqueador técnico para o usuário ter sucesso?"

**cortai-product:**
- Contexto: prioridades de produto baseadas em dados de retenção e ativação
- Métricas CEO para product: feature adoption, ativação D7, NPS por feature
- Pergunta-chave: "Qual feature mais aumenta Must Have %?"

**cortai-growth:**
- Contexto: objetivos de aquisição e retention para o quarter
- Métricas CEO para growth: MRR growth, CAC por canal, novos pagantes/semana
- Pergunta-chave: "Qual experimento de growth tem maior potencial esta semana?"

**cortai-finance:**
- Contexto: runway, burn rate, decisões de pricing
- Métricas CEO para finance: gross margin, runway, ARPU, LTV:CAC
- Pergunta-chave: "O que precisamos mudar para melhorar o Default Alive status?"
